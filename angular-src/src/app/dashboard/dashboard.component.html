News Analytics

In today's world AI has crept into the soul of all things important and the Media, is surely one thing that has been affected largely by it. By applying analytics on the plain old mundane NEWS, we could foray into a world of possibilities. This project is set to decipher a large contextual corpora of news analysis and be a major source of informed decisions. The Single page application is implemented on MEAN and draws intelligence from a python analysis engine.
